CÓMO CREAR OBSTÁCULOS MANUALES - PASO A PASO

1. CREAR UN OBSTÁCULO (ejemplo: fuente):
   - Crea un GameObject vacío
   - Renómbralo a "Fuente_Obstaculo"
   - Añade un Collider2D (BoxCollider2D o CircleCollider2D)
   - Configura el collider:
     * Is Trigger = FALSE (importante!)
     * Ajusta el tamaño para que coincida con la fuente

2. CONFIGURAR EL PERSONAJE:
   - Asegúrate de que tu personaje tenga:
     * Rigidbody2D
     * BoxCollider2D
     * Tag: "Player"

3. CONFIGURAR LAS CAPAS DE COLISIÓN:
   - Ve a Edit > Project Settings > Physics2D
   - En "Layer Collision Matrix":
     * Marca que "Default" colisione con "Default"
   - O crea una capa "Obstacles" y configura las colisiones

4. TIPOS DE COLLIDERS PARA DIFERENTES OBSTÁCULOS:

   FUENTE (circular):
   - CircleCollider2D
   - Radius: ajustar al tamaño de la fuente
   - Center: (0, 0, 0)

   BANCO (rectangular):
   - BoxCollider2D
   - Size: ajustar al tamaño del banco
   - Center: (0, 0, 0)

   ROCAS (irregular):
   - PolygonCollider2D
   - Usa "Edit Collider" para dibujar la forma

5. CONFIGURAR EL RIGIDBODY2D DEL PERSONAJE:
   - Body Type: Dynamic
   - Gravity Scale: 0
   - Freeze Rotation: true
   - Linear Drag: 0
   - Angular Drag: 0

6. CONFIGURAR LOS COLLIDERS DE OBSTÁCULOS:
   - Body Type: Static (para obstáculos fijos)
   - O Kinematic (para obstáculos que se pueden mover)

¡LISTO! Ahora el personaje chocará con los obstáculos y no podrá atravesarlos.

CONSEJOS:
- Usa "Edit Collider" para ajustar la forma exacta
- Los obstáculos deben tener Is Trigger = FALSE
- El personaje debe tener Rigidbody2D para que funcione la física
- Puedes hacer los colliders invisibles (sin SpriteRenderer)
