CONFIGURACIÓN DEL SISTEMA DE DISPARO

1. CONFIGURAR LA IMAGEN DEL PROYECTIL:
   - Ve a Assets/Characters/Sprites/power.png
   - Texture Type: Sprite (2D and UI)
   - Pixels Per Unit: 16
   - Filter Mode: Point (no filter)
   - Apply

2. CREAR EL PREFAB DEL PROYECTIL:
   - Arrastra power.png a la escena
   - Renombra a "Projectile"
   - Añade componentes:
     * Rigidbody2D (sin gravedad)
     * CircleCollider2D (como trigger)
     * Projectile (script)
   - Configura:
     * Tag: "Projectile"
     * Rigidbody2D: Gravity Scale = 0, Drag = 0
     * CircleCollider2D: Is Trigger = ✓
   - Arrastra a Assets/Characters/Prefabs/ para crear el prefab
   - Elimina de la escena

3. CONFIGURAR EL PERSONAJE PARA DISPARAR:
   - Selecciona tu personaje
   - Añade el componente "PlayerShooting"
   - Configura:
     * Projectile Prefab: Arrastra el prefab Projectile
     * Fire Point: Se crea automáticamente
     * Projectile Speed: 10
     * Fire Rate: 0.5 (tiempo entre disparos)
     * Use Right Click: ✓
     * Use Spacebar: ✗ (opcional)

4. CONFIGURAR TAGS NECESARIOS:
   - Ve a Edit > Project Settings > Tags and Layers
   - Añade estos tags si no existen:
     * "Player"
     * "Enemy" 
     * "Projectile"
     * "Obstacle"

5. PROBAR EL DISPARO:
   - Presiona Play
   - Mueve el mouse donde quieras disparar
   - Haz click derecho para disparar
   - El proyectil debería ir hacia el cursor

6. AJUSTAR CONFIGURACIÓN:
   - Si el proyectil va muy rápido: baja Projectile Speed
   - Si el proyectil va muy lento: sube Projectile Speed
   - Si disparas muy rápido: sube Fire Rate
   - Si disparas muy lento: baja Fire Rate

¡LISTO! Tu personaje ya puede disparar hacia el cursor con click derecho.

CONTROLES:
- Click derecho: Disparar hacia el cursor
- Barra espaciadora: Disparar (opcional)

CARACTERÍSTICAS:
- Dispara hacia donde apunta el mouse
- El proyectil se destruye al golpear algo
- Se destruye automáticamente después de 5 segundos
- No puede golpear al jugador que lo disparó
