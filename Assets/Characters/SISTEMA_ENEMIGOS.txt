📋 PASOS PARA CONFIGURAR:

1️⃣ CREAR PREFAB DEL ENEMIGO:
   - Arrastra Enemy.png a la escena
   - Configura como Sprite (2D)
   - Filter Mode: Point (no filter)
   - Agrega componentes:
     * Rigidbody2D (Gravity Scale: 0, Freeze Rotation: true)
     * CircleCollider2D (Is Trigger: false) ← IMPORTANTE: NO trigger
     * SpriteRenderer
     * Script Enemy.cs
   - Tag: "Enemy"
   - Guarda como prefab en Assets/Characters/Prefabs/

2️⃣ CONFIGURAR JUGADOR:
   - Asegúrate que el jugador tenga tag "Player"
   - Agrega CircleCollider2D (Is Trigger: true)

3️⃣ CREAR GAME MANAGER:
   - Crea GameObject vacío llamado "GameManager"
   - Agrega script GameManager.cs
   - Crea UI Canvas con:
     * Text para vidas: "Vidas: 10"
     * Text para Game Over: "GAME OVER!" (inicialmente oculto)

4️⃣ CREAR ENEMY SPAWNER:
   - Crea GameObject vacío llamado "EnemySpawner"
   - Agrega script EnemySpawner.cs
   - Asigna el prefab del enemigo
   - Configura: Total Enemies = 15

5️⃣ CONFIGURAR PROYECTIL:
   - Asegúrate que el prefab del proyectil tenga:
     * Tag: "Projectile"
     * CircleCollider2D (Is Trigger: true)
     * Script Projectile.cs

- ✅ Enemigos se mueven hacia el jugador
- ✅ Enemigos colisionan con obstáculos y otros enemigos

- ✅ Si enemigo toca jugador = -1 vida
- ✅ 10 vidas totales
- ✅ Game Over cuando se acaban las vidas
- ✅ Presiona R para reiniciar
